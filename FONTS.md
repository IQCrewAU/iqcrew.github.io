# Fonts — sources and licences

## General Sans (headings, body, wordmark)
Free under the ITF Free Font Licence (commercial use permitted; the licence doesn't allow us to redistribute the files, which is why they're not bundled here).

**Get it (2 minutes):** https://www.fontshare.com/fonts/general-sans → Download family → install Medium (500), Semibold (600), Bold (700). For web projects, use the woff2 files in the download's `WEB/fonts/` folder.

You do NOT need General Sans to use the logos — every logo file in this kit has text converted to outlines.

## JetBrains Mono (figures, labels, evidence lines)
Included in `fonts/jetbrains-mono/` — Regular + Medium, TTF + WOFF2, with its SIL Open Font Licence (OFL.txt). Redistribution is permitted, so it ships in the kit. Official source: https://github.com/JetBrains/JetBrainsMono

## Rules
- No substitutes, ever — including "close enough" system fonts.
- Web projects self-host both families (see the repo CLAUDE.md); no font CDNs in production.
