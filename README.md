## Which logo, when

- **Light background** (documents, website, most things): `-primary`
- **Dark or green background**: `-reverse`
- **Single-colour reproduction** (print, engraving, embroidery, watermarks): `-mono-ink` on light, `-mono-paper` on dark — letterforms knock out to transparent
- **Square slots** (app icons, avatars): monogram files, or the ready-made `social/` avatars
- **Email signatures**: `email/iqcrew-email-logo-600w.png` displayed at ~300 px wide
- **Tiny sizes** (<90 px wide): monogram only, never the full lockup

## Two hard rules before anything ships

1. **Record Red (#D9531E) exists only inside the stamp**, and the stamp goes only on verified artefacts — Scorecard reports, assessments, published case studies, the Trust Centre. Never on ads, the homepage, or anything unproven.
2. **Every public claim must be a PRINT NOW row in the claims register.** Design carries the words; it doesn't get to invent them.
